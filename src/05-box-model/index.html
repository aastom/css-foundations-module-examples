The CSS Box Model

Every HTML element is a box. The CSS Box Model describes how these boxes are rendered on a webpage.

Understanding the Components

Content Area (blue): This is where your actual content (text, images, video) resides.

Padding Area (light green): The space between the content and the border. It's inside the border.

Border Area (gray): The border that goes around the padding and content. You can style its width, style, and color.

Margin Area (light red): The space outside the border, separating the element from other elements.

box-sizing Property: content-box vs border-box

The box-sizing property allows you to define how the width and height of an element are calculated.

content-box (Default)

Width: 200px

Padding: 20px

Border: 5px

Actual Width: 200 + (20*2) + (5*2) = 250px

border-box (Recommended)

Width: 200px

Padding: 20px

Border: 5px

Actual Width: 200px
