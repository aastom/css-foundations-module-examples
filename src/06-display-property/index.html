The CSS display Property

The display property is fundamental to how elements behave and interact with other elements on a web page.

display: block

Block-level elements always start on a new line and take up the full width available. Examples include `div`, `p`, `h1`-`h6`.

display: inline

Inline elements do not start on a new line and only take up as much width as necessary. Examples include `span`, `a`, `img`.

This is a block element next to inline elements.

Inline elements do not respect `width`, `height`, `margin-top`, `margin-bottom` properties. They respect `padding` and `margin-left/right`.

display: inline-block

Inline-block elements behave like inline elements (they don't force a new line) but can also have `width`, `height`, and vertical `margin`/`padding` applied.

display: none

display: none completely removes an element from the document flow. It won't take up any space and is effectively invisible.
